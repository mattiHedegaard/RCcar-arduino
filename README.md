# Arduino RC Car

This project was developed as part of a 4th-semester course titled *Physical Prototypes and Interaction* in the Mediaology program at Aalborg University.
The purpose of the project was to address a problem statement through the development of an electronic prototype. This project explored a solution for people with limited mobility, allowing them to retrieve objects using an RC car.
The object retrieval functionality is currently theoretical and implemented using a controllable front-mounted hook operated through the controller.
I chose to design and model the RC car from scratch to further challenge myself throughout the project.

All programming, electrical wiring, and 3D modeling were created by me, with the exception of the differential model, which was downloaded from [Thingiverse](https://www.thingiverse.com/thing:6437946).

> Note: This repository was created after the project had already been developed, so there is limited commit history available.
